# Lucky Magisk Module (万吉网络神器 - 面具模块)

这是一个将 [gdy666/lucky](https://github.com/gdy666/lucky) 核心程序打包为 Magisk / KernelSU 模块的项目

lucky官网: https://lucky666.cn

通过本模块，你可以将任意一台已经 Root 的闲置安卓设备，化身为一台超低功耗、性能强悍的微型软路由/服务器。支持公网 IPv6 转内网 IPv4 端口转发、反向代理、DDNS、内网穿透、网络唤醒等强大功能

## ✨ 核心特性

本模块在官方核心程序的基础上，专为 Android Root 环境进行了深度定制，完美融入 Magisk/KernelSU 生态：

* 📦 **绿色独立，卸载即焚**：Lucky 的所有配置文件和运行日志均重定向至模块自身的 `data` 目录下 (`/data/adb/modules/lucky666/data/`)。不污染系统目录，卸载模块即可彻底清理。
* 🛡️ **进程守护，稳如老狗**：内置守护脚本。若 Lucky 进程意外崩溃，脚本会在几秒内自动将其拉起，确保网络服务 24 小时在线。
* 🎛️ **热插拔控制，实时启停**：**完美支持 Magisk/KernelSU 的模块开关控制！** 拨动开关10秒生效即可启动或关闭 Lucky 进程，**无需重启手机**。
* 📟 **UI 状态实时回显**：巧妙利用属性动态写入技术。在面具管理界面下拉刷新，即可在模块描述中实时查看 Lucky 的运行状态（🟢运行中 / 🔴已停止）及进程 PID。

## 📥 安装指南

1. 在本项目的 [Releases] 页面下载最新版本的 `lucky.zip` 模块。
2. 打开 Magisk Manager 或 KernelSU。
3. 进入“模块”界面，点击“从本地安装”。
4. 选择下载的 `lucky.zip` 文件，等待刷入完成。
5. 重启手机。

## 🚀 使用方法

### 1. 访问后台界面
重启手机并确保连接到局域网。在同一局域网下的电脑或其他设备浏览器中输入：
* **管理地址**: `http://<手机局域网IP>:16601`
* **默认账号**: `666` 
* **默认密码**: `666` 

*登录后请务必第一时间修改默认账号和密码！*

### 2. 终端高级维护指令 (进阶)
如果你忘记了后台密码，或者需要紧急操作，可以在手机上下载 [Termux](https://termux.dev/)，输入 `su` 获取 Root 权限后，直接执行以下命令：

* `lucky -rResetUser`：重置用户账号密码为 666:666 
* `lucky -rRestart`：强制重启 lucky 服务 
* `lucky -rUnlock`：立即解锁登录限制 
## 📁 模块目录结构

```text
/data/adb/modules/lucky666/
 ├── module.prop       # 模块信息与动态状态回显
 ├── service.sh        # 核心守护脚本 (处理开机自启、状态更新、开关监听)
 ├── data/             # (运行后自动生成) 存放配置 lucky.conf 和 run.log
 └── system/
      └── bin/
           └── lucky   # 核心二进制可执行文件
```
